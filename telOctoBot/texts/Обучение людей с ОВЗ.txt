Обучение людей с ОВЗЯрГУ им. П.Г. Демидова обеспечивает условия для инклюзивного обучения лиц с ограниченными особенностями здоровья. Для этого на этапах поступления, обучения и трудоустройства ведется специализированный учет студентов с ОВЗ, с письменного согласия студентов создается особая база данных, содержащая информацию о здоровье обучающихся.Для абитуриентов с ОВЗ существует система довузовской подготовки, включающая в себя проведение профориентаций, дней открытых дверей, консультаций по вопросам приема и поступления, организацию участия абитуриентов в вузовских олимпиадах школьников. Кроме того, ЯрГУ обеспечивает проведение вступительных испытаний в особой форме – абитуриент может выбрать форму испытания (устная/письменная), использовать технические средства, для него может быть увеличено время сдачи испытания. Для этого нужно написать заявление.Большинство зданий университета оснащено удобным входом, доступным для людей с нарушениями слуха, зрения, опорно-двигательного аппарата. Аудитории вуза также приспособлены для обучения по средствам наличия аудио-/видеотехники, позволяющей обеспечить разные форматы восприятия информации студентами.Кроме того, для студентов с ОВЗ может быть адаптирован учебный план (в соответствии с нарушениями). Возможно введение специальных предметов для коррекции нарушений коммуникации, адаптации студентов в вузе (предметы вводятся по необходимости), устанавливается особый порядок прохождения дисциплины «физическая культура». Продолжительность обучения может быть увеличена, но не более, чем на год (по необходимости).Для студента может быть организовано дистанционное обучение, которое обеспечивается только после письменного обращения студента после оценки целесообразности такой формы обучения. Для обучения студентов с нарушениями слуха и зрения ЯрГУ может пригласить сурдопереводчика или тифлопедагога.Итоговая и промежуточная аттестации для студентов с ОВЗ проводятся в соответствии с их потребностями и возможностями, с учетом психофизических особенностей.Во время обучения обучающемуся организуется практика, подходящая ему по психофизическим особенностям.
